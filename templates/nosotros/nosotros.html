¿Quienes somos?

Somos un aplicativo web desarrollado con tecnologías HTML, CSS, Django y SQL Server, creado como parte de un proyecto académico orientado a la restitución y reparación de tierras. Este desarrollo integra conocimientos de asignaturas clave como Gestión de Bases de Datos, Estructura de Datos, Seguridad de Aplicaciones y Programación Web. El sistema fue diseñado y elaborado por Felipe Pinilla, Sara Rivera y Ronald Pérez, con el objetivo de ofrecer una herramienta digital que sirva tanto para la investigación básica como para la gestión de solicitudes reales relacionadas con la restitución de tierras. El aplicativo permite a cualquier usuario acceder a información investigativa sin necesidad de registrarse. Sin embargo, si un usuario cuenta con un registro válido en la base de datos (Es decir, si es una víctima del conflicto), puede generar una solicitud. Dicha solicitud será evaluada por un administrador, quien validará su veracidad y completitud. Si todo está en orden, se convierte en un caso activo y se asigna a un abogado, quien procederá a iniciar el proceso legal correspondiente. Este sistema busca ser un canal de apoyo real para las víctimas, al mismo tiempo que ofrece una plataforma informativa para el resto de los usuarios.
